## Development

You will need:

- `ruby`
- `node`

Before proceeding install globally:

- `jekyll` ruby gem
- `yarn` node package

Then run:

- `yarn install`
- `npx gulp`

You are all set!


## Releasing

Run `npx gulp build` to make a full build.
